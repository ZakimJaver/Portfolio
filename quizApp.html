Quiz App

Log In

Users are greeted with a login page. New users will need to sign up before starting the quiz.

Sign Up

Here we created a new user with a username and password. We can also chagne the theme by clicking on the three dots in the menue bar. Users can change the theme at any time on any screen

Log In 2

The theme stays for all the pages. Here we can log in with the user we created. If the details are wrong it would show use an error message

The Quiz

Once logged-in the use can start the test. The questions are shown randomly and the bottom show the users score

End Of Quiz

When the test is completed the user is taken to the score page where they can see their score. If the user achieves a high score, an image is shown (new high score). Users could try again or clear their score

High Score Page

If the user wants to see other users high score they need click on the “HIGH SCORE” on the top of the menu bar. To log out they simply click the logout button on the top menu bar
