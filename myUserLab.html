Admin App

Home Page

The admin is given the choice to either manage the users or the questions. The data we mange is from the Quiz Apps database and all changes will be see on the Quiz App.

Users

In the users page we are able to add a users or edit an existing user. By pressing 'DISPLAY USERS' we are able to view a list of all the current users.

Edit User

To edit a user we would press on the user we would like to edit and either delete the user or edit their details.

Add User

To add a user we would press the 'ADD USER' and fill in the users details then press 'ADD NEW USERS' to add it to the database.

Questions

Managing the questions is exactly the same as manageing the users.
